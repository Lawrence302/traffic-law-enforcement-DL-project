About

This project is a computer vision application designed to enhance road safety by analyzing images and videos for helmet compliance and rider behavior. Using advanced object detection algorithms powered by OpenCV and related technologies, the system can:

  • Identify and annotate riders, bikes, helmets, and instances of non-compliance (e.g., no helmet).
  • Automatically crop and save regions of interest (ROIs) where violations occur for further analysis or reporting.
  • Store and organize detailed information about detected violations, including annotated images and metadata.

The goal of this project is to leverage computer vision techniques to support road safety initiatives, enabling automated monitoring and actionable insights for traffic law enforcement and public awareness campaigns.

Key Features:

  • Support for both video and image inputs.
  • Real-time detection and annotation of multiple objects within a frame.
  • Scalable and efficient processing pipeline for large datasets.

This project is an example of how artificial intelligence and machine learning can be applied to solve real-world challenges, ensuring better compliance and safety on roads.
